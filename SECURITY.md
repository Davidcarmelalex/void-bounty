# Security Policy — VOID.Bounty

This platform handles vulnerability reports. The irony of having a security policy here is not lost on us.

## Reporting
Report via [GitHub Security Advisories](../../security/advisories/new). Response within 24h.

## Scope
- Authentication bypass on report submission
- Unauthorized program access or data exfiltration
- AI triage manipulation (inflating severity)
- Hunter identity exposure
- SQL injection in report content
